# 🤖 AI Console Assistant

Умный консольный помощник для Linux, работающий с локальным сервером **Ollama**. 
Ассистент генерирует bash-команды по вашему описанию, объясняет их действие и позволяет выполнить их одним нажатием Enter.

## ✨ Особенности

*   **Генерация команд:** Превращает естественный язык ("как найти большие логи") в готовые bash-команды.
*   **Безопасность:** Показывает команду и пояснение *перед* выполнением. Вы можете отредактировать её или отменить запуск.
*   **Умный выбор моделей:** Автоматически определяет, какая модель лучше подходит для задач (код или чат). Поддерживает переключение между моделями "на лету".
*   **Гибкая настройка:** Управление адресом сервера Ollama и выбором моделей через ключи командной строки.
*   **Локальность:** Все данные обрабатываются на вашем сервере, ничего не уходит в облако.
*   **Редактирование на лету:** Встроенный редактор позволяет поправить команду перед запуском (например, изменить дату или путь).

## 🚀 Установка

### Требования
*   Linux (Debian, Ubuntu, AstraLinux, Armbian и др.)
*   Установленный и запущенный сервер [Ollama](https://ollama.com/)
*   Утилиты: `curl`, `jq`

### Быстрая установка

1.  Скачайте скрипт и сделайте его исполняемым:
    ```bash
    sudo curl -L https://raw.githubusercontent.com/sirius-parallax/ai-assist/main/ai -o /usr/local/bin/ai
    sudo chmod +x /usr/local/bin/ai
    ```
    *(Замените ссылку на актуальную после публикации)*

2.  Установите зависимости (если нет):
    ```bash
    sudo apt install curl jq
    ```

3.  Проверьте установку:
    ```bash
    ai -h
    ```

## ⚙️ Настройка

При первом запуске скрипт создаст конфиги в домашней директории (`~/.ai_assistant.conf` и `~/.ai_config`).
